Introduction

The Multipurpose AI Solution API is a suite of RESTful JSON endpoints that are specifically designed to meet the mission-critical demands of application developers, data scientists, and enterprise business platforms. This API reference includes all technical documentation developers need to integrate third-party applications and platforms.

Endpoint Overview

The Multipurpose AI Solutions API has 1 top level entry point:

Endpoint Description Example Call
/canopus/ This is the URL path which will be present in all the API calls. www.example.com/canopus/{endpoint here}

Top level entry point allows 4 additional API categories:

Endpoints Description Example Call
/deep-dreams Functionality here www.example.com/canopus/deep-dreams/
/style-transfer Functionality here www.example.com/canopus/style-transfer/
/image-classification Functionality here www.example.com/canopus/image-classification/
/face-recognition Functionality here www.example.com/canopus/face-recognition/
